How It Works

From idea to working app in four simple steps

1

Write Your Idea

Create a markdown file describing your app. Include features, tech stack preferences, and any specific requirements. Drop it in ~/app-ideas/ideas/.

2

One-Click Planning

Click "Run Planning" and watch Claude analyze your idea. It creates a comprehensive IMPLEMENTATION_PLAN.md with architecture, file structure, and step-by-step tasks.

3

Build with Stories

Generate user stories from your plan, then implement them one by one. Or skip stories and run full implementation in one shot‚Äîyour choice.

4

Launch & Push
